# Sistema de Login Completo em JavaScript

Este projeto cria um sistema de login simples e funcional usando:

* Frontend: HTML + CSS + JavaScript
* Backend: Node.js + Express
* Banco de dados: SQLite
* Senhas criptografadas com bcrypt
* Login com JWT

Você só precisa copiar os arquivos exatamente como estão.
